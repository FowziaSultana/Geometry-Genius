What is DOM and what is the purpose of DOM?

The Document Object Model, commonly referred to as the DOM, is a programming interface used for web documents. The DOM presents the document as a series of nodes and objects, enabling programming languages to interact with the page. It enables JavaScript to alter its structure, style, and content. by creating a tree-like representation of the HTML document, called the Document Object Model, and defining the ways in which that tree can be accessed.

How will you select HTML elements using DOM? Name the DOM methods.

To select HTML elements using the Document Object Model (DOM), you can use various methods. Here are some commonly used methods:

  1. getElementById(): This method selects an element by its ID attribute.
  2. getElementsByClassName(): This method selects elements that have a specific class name
  3. getElementsByTagName(): This method selects elements that have a specific HTML tag name.
  4. querySelector(): This method selects the first element that matches a specified CSS selector.
  5. querySelectorAll(): This method selects all elements that match a specified CSS selector.

What is event bubble?

In event bubbling, an event is first triggered on the target element, and then it "bubbles up" through its ancestors in the DOM hierarchy until it reaches the outermost DOM element or the document object. This means that if an event is triggered on a child element, it will also trigger on all of its parent elements, in order of their nesting hierarchy.Event bubbling is the default behavior in most web browsers, but it can be stopped or prevented by using the stopPropagation();

What is a callback function and why will you use it?

When a function is passed as an argument into another function and is invoked inside that outer function it is called a callback function. Callback functions are used when one function must execute only after another function has finished executing.
